404

Page Not Found? What does that even mean?

This page is showing you a specific type of error that just happened.
The error has a numerical code, or "HTTP status code" - this one is a "404 Page Not Found Error".


An HTTP status code is how the Web server communicates to your Web browser what has happened.

To understand the message being communicated, you have to look up the code number.

I remember the first time I realized this was actual information I could use to solve an internet problem and not just a confusing annoyance.
I'm no HTTP code wizard, but I can google a code and get at least some idea of what is wrong and then use that to figure out what the next steps are.
You should give it a try!

This is a good place to start, & here's another good resource.
